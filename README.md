# Ragnar Revival

A ZX Spectrum 48K 3D isometric game originally developed in the 1980s in Z80 assembly language.

## About This Project

Ragnar Revival was originally developed as a 3D isometric demo for the Sinclair ZX Spectrum 48K. The demo showcased an 8x8 block rendering system with 2 layers, written entirely in Z80 assembly language.

### Historical Context

This project represents early experiments with 3D isometric graphics on the ZX Spectrum, pushing the boundaries of what was possible on a home computer with just 48KB of memory.
